# User-Level Claude Configuration

이 파일은 `~/.claude/CLAUDE.md`에 위치하며, 모든 프로젝트에 적용됩니다.

## 개인 정보

- **이름**: Nill
- **역할**: UI/UX 디자이너 & 서비스 기획자
- **선호 언어**: 한국어

## 작업 스타일

### 선호하는 응답 방식
- 간결하고 실용적인 답변
- 코드 예시 포함
- 한국어로 응답

### 자주 사용하는 기술 스택
- Frontend: React, Next.js, TypeScript
- Styling: Tailwind CSS
- Backend: Supabase, Node.js
- Design: Figma

## 기본 규칙

### 코드 작성 시
1. TypeScript 필수 사용
2. 함수형 컴포넌트 선호
3. TailwindCSS 클래스 사용
4. 한국어 주석 허용

### 커밋 메시지
- 한국어 또는 영어 (혼용 가능)
- Conventional Commits 형식

## 자주 사용하는 명령어

```
/spec-generate  - 기능 명세서 생성
/wireframe      - 와이어프레임 생성
/design-review  - 디자인 리뷰
/flow-check     - 플로우 검증
```

## 비활성화할 도구

특정 상황에서 비활성화:
- 간단한 질문: 웹 검색 비활성화
- 로컬 작업: 클라우드 MCP 비활성화

## 개인 단축키

```
"기획" → /spec-generate 실행
"와이어" → /wireframe 실행
"리뷰" → /design-review 실행
```
