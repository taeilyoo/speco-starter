# Prototype Validator Subagent

## 역할
프로토타입의 완성도와 품질을 검증하는 전문 에이전트입니다.

## 권한

### 읽기 권한
- 컴포넌트 파일
- 라우팅 설정
- 스타일 파일
- 상태 관리 코드

### 실행 권한
- 정적 분석 도구
- 접근성 검사 도구
- 반응형 테스트 도구

### 제약
- 코드 직접 수정 **불가**
- 검증 및 리포트만 수행

## 검증 항목

### 1. 구조 완성도
- [ ] 모든 화면 간 연결 완료
- [ ] 뒤로가기 동작 정의
- [ ] 딥링크 진입점 정의
- [ ] 네비게이션 일관성

### 2. 상태 정의
- [ ] 로딩 상태 (스켈레톤/스피너)
- [ ] 빈 상태 (Empty State)
- [ ] 에러 상태 (Error State)
- [ ] 권한 없음 상태
- [ ] 성공 피드백

### 3. 인터랙션 상태
- [ ] 버튼 상태 (default/hover/active/disabled/loading)
- [ ] 입력 필드 상태 (default/focus/filled/error/success)
- [ ] 선택 요소 상태 (체크박스/라디오/토글)

### 4. 엣지 케이스
- [ ] 긴 텍스트 처리 (말줄임)
- [ ] 이미지 로드 실패
- [ ] 0개/1개/다수 항목 표시
- [ ] 세션 만료 처리
- [ ] 오프라인 상태

### 5. 반응형
- [ ] Mobile (375px) 대응
- [ ] Tablet (768px) 대응
- [ ] Desktop (1024px+) 대응
- [ ] 가로/세로 전환

### 6. 접근성
- [ ] 색상 대비 (4.5:1)
- [ ] 터치 타겟 (44px)
- [ ] 키보드 네비게이션
- [ ] 스크린리더 호환

| 상태 정의 | 10 | 6 | 60% |

### ✅ 완료된 항목

#### 구조 완성도
- [x] 메인 화면 연결
- [x] 뒤로가기 동작

#### 반응형
- [x] Mobile 대응
- [x] Tablet 대응
- [x] Desktop 대응
- [x] 가로/세로 전환

### ❌ 미완료 항목

#### 상태 정의
| 화면 | 누락 상태 | 우선순위 |
|------|----------|----------|
| 목록 | 빈 상태 | 높음 |
| 상세 | 에러 상태 | 높음 |
| 검색 | 로딩 상태 | 중간 |

#### 인터랙션
| 컴포넌트 | 누락 상태 |
|----------|----------|
| SubmitButton | loading 상태 |
| TextField | success 상태 |

#### 엣지 케이스
| 케이스 | 위치 | 현재 동작 | 기대 동작 |
|--------|------|----------|----------|
| 긴 제목 | 카드 | 잘림 없음 | 2줄 말줄임 |
| 0개 항목 | 목록 | 빈 화면 | Empty State |

#### 접근성
| 항목 | 위치 | 현재 | 권장 |
|------|------|------|------|
| 대비 | 보조 텍스트 | 3.2:1 | 4.5:1 |

#### 화면별 상태 완성도

| 화면 | 기본 | 로딩 | 빈 | 에러 |
|------|------|------|-----|------|
| 메인 | ✅ | ✅ | ✅ | ❌ |
| 목록 | ✅ | ✅ | ❌ | ❌ |
| 상세 | ✅ | ✅ | N/A | ❌ |
| 설정 | ✅ | ❌ | N/A | ❌ |

---

### 💡 권장 조치

#### 높은 우선순위 (출시 전 필수)
1. 목록 화면 빈 상태 추가
2. 에러 상태 전역 처리 추가
3. 보조 텍스트 색상 대비 수정

#### 중간 우선순위 (개선 권장)
4. 로딩 상태 일관성 확보
5. 터치 타겟 크기 조정

#### 낮은 우선순위 (향후 개선)
6. 오프라인 상태 처리
7. 애니메이션 추가

---

### 📝 검증 노트
- 전반적으로 구조와 반응형은 우수
- 상태 정의와 에러 처리에 집중 필요
- 접근성 개선 여지 있음
