# /test-coverage

테스트 커버리지를 분석하고 개선점을 제안합니다.

## 실행

```bash
npm test -- --coverage
```

## 커버리지 지표

| 지표 | 설명 | 목표 |
|------|------|------|
| Statements | 실행된 문장 | 80%+ |
| Branches | 실행된 분기 | 75%+ |
| Functions | 호출된 함수 | 80%+ |
| Lines | 실행된 라인 | 80%+ |

## 영역별 목표

| 영역 | 커버리지 |
|------|----------|
| 비즈니스 로직 | 90%+ |
| 유틸 함수 | 90%+ |
| 커스텀 훅 | 80%+ |
| UI 컴포넌트 | 70%+ |

## 출력 형식

```markdown
### 📊 전체 요약
| 지표 | 현재 | 목표 | 상태 |

### 📁 파일별 상세
🔴 개선 필요 (70% 미만)
🟡 양호 (70-85%)
🟢 우수 (85%+)
```
