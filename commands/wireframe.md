# /wireframe

화면 구조를 텍스트 기반 와이어프레임으로 생성합니다.

## 실행 단계

1. 레이아웃 구조 설계
2. 컴포넌트 명세
3. 상태별 화면 (기본/로딩/빈/에러)
4. ASCII 와이어프레임 생성

## ASCII 요소 가이드

```
버튼:        [ Button Text ]
입력 필드:   [________________]
체크박스:    [x] 선택됨  [ ] 미선택
드롭다운:    [선택하세요       ▼]
스켈레톤:    ░░░░░░░░░░░░
```
