# /a11y-audit

접근성(Accessibility) 전문 검사. WCAG 2.1 AA 기준.

## 검사 영역

### 1. 시각적 접근성
- 색상 대비 (4.5:1 / 3:1)
- 색상 의존성 검사

### 2. 키보드 접근성
- Tab 접근 가능 여부
- 포커스 표시 명확성

### 3. 스크린리더 호환
- alt 텍스트, aria-label
- heading 계층 구조

### 4. 모터 접근성
- 터치 타겟 44x44px
- 제스처 대안

## 출력 형식

```markdown
## 접근성 검사 결과

### 📊 준수율 요약
### 🔴 Level A 위반 (필수)
### 🟡 Level AA 위반 (권장)
```
