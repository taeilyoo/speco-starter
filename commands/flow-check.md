# /flow-check

사용자 플로우의 완성도와 논리적 일관성을 검증합니다.

## 실행 단계

### 1. 진입점 분석
- 앱 아이콘, 딥링크, 푸시 알림 진입

### 2. 연결성 검증
- Dead End, 무한 루프, 고아 화면 탐지

### 3. 이탈 지점 분석
- 3클릭 이상 소요 기능 식별

### 4. 엣지 케이스 점검
- 뒤로가기, 세션 만료, 권한 거부 처리

## 출력 형식

```markdown
## 플로우 분석 결과

### 🗺️ Mermaid 플로우 다이어그램
### 🔴 Critical / 🟡 Warning
| 위치 | 문제 유형 | 설명 |
```
