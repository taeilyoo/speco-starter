# /design-review

UI/UX 일관성과 디자인 품질을 종합 검토합니다.

## 실행 단계

### 1. 디자인 시스템 준수 검사
- 색상 토큰 사용 여부
- 타이포그래피 규격 준수
- 간격 시스템 (4px 그리드) 준수

### 2. 컴포넌트 일관성 검사
- 버튼/입력 필드 상태 정의 완성도
- 컴포넌트 재사용성 평가

### 3. 접근성 검사 (WCAG 2.1 AA)
- 색상 대비 비율 (4.5:1)
- 터치 타겟 크기 (44x44px)
- aria 속성 존재 및 적절성

### 4. 반응형 검사
- 브레이크포인트별 레이아웃

## 출력 형식

```markdown
## 디자인 리뷰 결과

### 📊 요약
- 디자인 시스템 준수율: ___%
- 접근성 준수율: ___%

### 🔴 Critical / 🟡 Warning / 🟢 Info
| 위치 | 문제 | 권장 조치 |
```
